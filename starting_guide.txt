## 🚀 Quick Start

### Prerequisites
- Python 3.10 (NOT 3.12/3.13)
- ~2GB disk space for dataset

### One-Command Setup
```bash
py -3.10 -m pip install -r requirements.txt
py -3.10 run_all.py

# 1. Install dependencies
py -3.10 -m pip install -r requirements.txt

# 2. Download fresh dataset (1000 imgs × 8 animals)
py -3.10 download_fresh.py

# 3. Remove any duplicates
py -3.10 fix_duplicates.py

# 4. Train teacher + distill student
py -3.10 distill.py

# 5. Convert for ESP32-CAM
py -3.10 convert.py

# 6. Generate proof-of-work PDF
py -3.10 generate_report.py

cd Animal_detection
py -3.10 --version

py -3.10 convert.py
py -3.10 run_all.py


File	                        What It Does	                        When to Run
download_fresh.py	            Downloads 1000 images per animal	    Once (or when you want fresh data)
fix_duplicates.py	            Removes duplicate images	            After downloading
distill.py	                    Trains teacher + distills student	    After dataset is ready
convert.py	                    Makes TFLite + C header for ESP32	    After training
generate_report.py	            Creates proof PDF	                    Before submission
demo.py	                        Live terminal demo	                    For demo/video
run_all.py	                    Runs everything in order	            Lazy mode 😄
config.py	                    All settings in one place	            Edit when changing config
